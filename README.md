## 🛠️ Setup & Execution

### 1️⃣ Prerequisites

Ensure you have an active local server environment (e.g., XAMPP, LAMP) or the PHP CLI installed.

* 📥 [Download XAMPP](https://www.google.com/search?q=https://www.apachefriends.org/index.html)
* Verify PHP installation:
```bash

```



php -v

```

### 2️⃣ Execution
Scripts can be executed via a traditional web server setup (`htdocs` / `/var/www/html`) or through the built-in PHP development server.

```bash
# Navigate to the specific lab date directory
cd PHP-LAB/UNIT1/05042026

# Initiate the built-in PHP development server
php -S localhost:8000

```

Access the output in your browser at `http://localhost:8000`.
